# Coffee & Clarity Website

A polished, free-hosting-ready static website for Coffee & Clarity.

## Pages/Sections
- Home
- Services
- Schedule with Calendly embed

## How to preview
Open `index.html` in your browser.

## How to publish free
### GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and the `assets` folder.
3. Go to Settings → Pages.
4. Choose Deploy from branch → main → root.
5. Save.

### Netlify
1. Go to Netlify.
2. Drag this entire folder into Netlify Drop.
3. Netlify will give you a free live website link.
